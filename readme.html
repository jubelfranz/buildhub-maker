🛠️ BuildHub Maker Setup Guide

Welcome to your professional build and deployment environment.

Important: This plugin contains sensitive credentials. Never push your maker-config.php to a public repository!

🚀 Quick Start

Step 1: Installation
Upload and activate the plugin. You will see a red warning notice until the configuration is found.
Step 2: Configuration
Navigate to the plugin folder and locate the templates directory. Copy template-maker-config.php to the root folder and name it maker-config.php.
Step 3: GitHub Secrets
Go to your GitHub Repository > Settings > Secrets > Actions. Add the following secrets:
  • WPORG_USERNAME & WPORG_PASSWORD
  • MAIL_USERNAME & MAIL_PASSWORD

📂 Folder Structure

/buildhub-maker/
├── buildhub-maker.php    (Main Loader)
├── maker-config.php      (Your Credentials)
├── core/                (Transformation & Deployer)
├── admin/               (User Interface)
└── templates/           (Config Template)
